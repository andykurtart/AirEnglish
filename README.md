# ✈️ Travel Guide PWA

Справочник английских фраз и гайдов для путешественников. **Работает полностью офлайн** — идеально для использования в самолёте.

## Возможности

- 💬 **150+ фраз** на английском с переводом на русский по 6 категориям
- 📖 **6 пошаговых гайдов** (регистрация на рейс, таможня, поведение на борту, отель, трансфер, задержки рейсов)
- 🔖 **Избранное** — сохраняйте нужные фразы
- 🔍 **Поиск** по всем фразам
- 🌙 **Тёмная тема** — авто или вручную
- 📲 **PWA** — добавьте на домашний экран iPhone/Android
- ✈️ **Офлайн** — всё работает без интернета

## Категории фраз

| Категория | Фразы |
|-----------|-------|
| ✈️ Аэропорт | Регистрация, паспортный контроль, таможня |
| 🛫 На борту | Сиденья, питание, безопасность |
| 🚌 Трансфер | Такси, транспорт, навигация |
| 🏨 Отель | Заезд, сервис, выезд |
| 🆘 Экстренные | Помощь, медицина, документы |
| 💬 Основные | Приветствия, вежливость, базовые фразы |

## Деплой на GitHub Pages

1. Создайте репозиторий на GitHub
2. Загрузите все файлы (индексируйте структуру ниже)
3. Перейдите в **Settings → Pages → Source: main branch, root folder**
4. Ваш сайт будет доступен по адресу `https://username.github.io/repo-name/`
5. На iPhone: откройте в Safari → **Поделиться → На экран «Домой»**

## Структура файлов

```
travel-guide/
├── index.html          ← Основное приложение
├── manifest.json       ← PWA манифест
├── sw.js               ← Service Worker (офлайн)
├── data/
│   └── phrases.js      ← Все фразы и гайды
└── icons/
    ├── icon-192.png    ← Иконка PWA
    └── icon-512.png    ← Иконка PWA (большая)
```

## Использование офлайн

После первого посещения страницы Service Worker кэширует все данные. При следующих открытиях (даже в самолёте без интернета) приложение работает полностью из кэша.
